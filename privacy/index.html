プライバシーポリシー

RT-Lab は、放射線技師向けのWebツールサイトとして、入力データをできる限りブラウザ内で処理し、サーバー保存を前提としない構成を採用しています。

1. 入力データの扱い

DRL比較ツール、MTF計算ツール、NPS計算ツール、造影CTシミュレータで扱う画像、DICOM、CSVなどの入力データは、原則としてブラウザ内で処理されます。サイト運営者が内容を収集・保存することを目的とした設計ではありません。

2. アクセス時に外部へ接続されるサービス

  • トップページでは、表示用フォント読込のため Google Fonts を利用する場合があります。
  • 一部のツールページでは、グラフ描画やCSV読込のため jsDelivr CDN 配信のJavaScriptライブラリを利用しています。
  • サイトは GitHub Pages を用いて公開されています。

これらのサービスでは、アクセス時にIPアドレスやUser-Agentなどの通信上必要な情報が各配信元へ送信される可能性があります。

3. Cookieと解析

2026年3月27日時点で、RT-Lab は独自のアクセス解析ツールや広告配信を導入していません。

4. 安全管理

患者情報や施設情報を含むデータを扱う場合は、利用者自身の責任で匿名化や院内ルールへの適合を確認してください。RT-Lab は医療機器ではなく、計算結果は参考値です。

5. お問い合わせ

本ポリシーや掲載内容に関するお問い合わせは、GitHub Issues から受け付けています。
